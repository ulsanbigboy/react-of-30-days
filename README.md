## 👩‍🏫 사용법

독립 실행형 React 애플리케이션을 실행하는 단계 (01~12일차):

```bash
Open the `index.html` file in your favourite browser.
```
Webpack 기반 React 애플리케이션을 실행하는 단계 (13~30일차):

```bash
# install the dependencies
yarn install

# start the project
yarn start
```
